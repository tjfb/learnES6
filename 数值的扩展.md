# 二进制、八进制

ES6 提供了二进制和八进制数值的新的写法，分别用前缀0b（或0B）和0o（或0O）表示。
如果要将0b和0o前缀的字符串数值转为十进制，要使用Number方法。

    Number('0b111')  // 7
    Number('0o10')  // 8

# 数值分隔符

ES2021，允许 JavaScript 的数值使用下划线（_）作为分隔符。
为了编码方便

Number()、parseInt()、parseFloat()方法不支持
Number(123_123)
parseInt(123_123)
